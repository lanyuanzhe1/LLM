# 讯飞星辰 Agent 开发指南

> 官方文档：https://www.xfyun.cn/doc/spark/Agent03-开发指南.html
>
> 平台入口：https://agent.xfyun.cn

## 智能体类型

| 类型 | 适用场景 | 开发门槛 |
|------|----------|----------|
| **指令型智能体** | 文案生成、翻译、故事创作、AI画图 | 零代码，一句话创建 |
| **工作流智能体** | 智能客服、知识问答、复杂业务编排 | 低代码，拖拽式可视化 |
| **全自主开发** | 深度定制业务 | 需编程能力 |

## 工作流节点类型（20 种）

| 类别 | 节点 |
|------|------|
| **基础** | 开始、结束、大模型、代码、知识库 |
| **工具** | 工具节点、MCP节点、RPA节点 |
| **逻辑** | 决策节点、分支器、迭代节点、Agent智能决策节点 |
| **转换** | 变量存储器、变量提取器、文本处理 |
| **高级** | 消息节点、问答节点、知识库Pro、工作流节点、数据库节点 |

## 知识库节点

| 参数 | 说明 |
|------|------|
| Query | 检索关键词（输入） |
| Top K | 召回数量，默认 3 |
| Score 阈值 | 匹配度下限 |
| 输出 | `result(Array<Object>)` |

**知识库Pro节点**：Agentic RAG + Long RAG 双重策略，支持多意图理解和跨知识点关联。

分段策略：智能分段（默认）或自定义分段（标识符 + 段落长度 200-300 字）。

## 大模型节点

| 参数 | 说明 |
|------|------|
| 模型选择 | 官方模型 + 个人接入模型 |
| 最大回复长度 | 默认 2048 Token |
| 核采样阈值 | (0,1] |
| 生成多样性 | 1-6 |
| 联网搜索 | DeepSeek-V3/R1、星火128k等支持 |

## 发布渠道

| 渠道 | 条件 |
|------|------|
| 星火Desk/APP | 直接发布 |
| API | 实名认证 → 绑定APPID → 获取 Key |
| MCP Server | 填写服务名称、概述、content |
| 微信公众号 | 服务号对接 |

免费额度：指令型 1000万 token，工作流 500万 token。每位用户最多 100 次免费 API 调用。
