## 注册与创建应用

1. 打开 https://www.xfyun.cn/ → 右上角登录/注册（微信扫码或手机号）
2. 进入「控制台」→「我的应用」→「创建新应用」
3. 填写应用名称、分类、功能描述 → 提交

## 三大凭据

| 凭据 | 说明 |
|------|------|
| **APPID** | 应用唯一标识（公开） |
| **APIKey** | API 访问密钥（控制台可见） |
| **APISecret** | API 密钥（仅首次显示，需立即保存！遗忘只能重置） |

当前项目凭据：APPID=`5c75015a`，APIKey 和 APISecret 已保存。

## 鉴权方式区分

不同 API 使用不同鉴权算法：

| API | 鉴权方式 | 关键词 |
|-----|---------|--------|
| **Embedding API** | HMAC-SHA256 | `host date request-line digest` 签名原文 |
| **ChatDoc API** | MD5 + HmacSHA1 | `MD5(appId+timestamp)` → `HmacSHA1` → `Base64` |
| **星火大模型 API** | WebSocket URL 签名 | 与 Embedding 类似但走 WSS |

## 开通服务

不同服务需**分别开通**：
- Embedding → 控制台 → 服务管理 → 搜索"Embedding"
- 星火大模型 → 控制台 → 服务管理 → 搜索"星火"
- ChatDoc → 同上

每个服务有独立的 API 配额和计费。
