## 数据集概述

面向粮食储藏（粮食储藏）垂直领域大模型的**指令微调（SFT）**数据集，包含 422 条高质量 instruction-input-output 三元组。所有问答对基于专业知识手工编写，覆盖 8 个核心子领域。

## 数据格式

```json
{
  "instruction": "你是一位粮食储藏领域的资深专家...（系统提示）",
  "input": "储粮害虫的物理防治方法有哪些？",
  "output": "储粮害虫的物理防治方法主要包括：\n\n1. 高温杀虫...",
  "system": "（同 instruction）",
  "history": []
}
```

## 数据特点

- **专业准确**：每条输出均使用领域规范术语，涉及标准时注明具体条款编号
- **问法多样**：通过同义改写扩充用户问法（正式提问、口语简短、限定条件追问等）
- **输出结构化**：答案按要点分层，便于模型学习专业回答的组织逻辑
- **完整覆盖**：17 篇源文档中 14 篇覆盖率超过 70%，核心文档覆盖率 100%

## 适用任务

- 指令微调（SFT / LoRA）
- 目标模型：Qwen-2.5-7B-Instruct 等同规模中文大模型
- 平台：讯飞星辰 MaaS、LLaMA-Factory、Hugging Face Trainer 等
